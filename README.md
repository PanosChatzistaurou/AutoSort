# AutoSort

Installation & Execution

The source code in this repository cannot be run directly. Use the pre-compiled release:

    Download: Get the latest executable from the Releases page.

    Placement: Place the executable in the same directory as the local_nlp_model folder.

    Run: Execute the file from that directory.
